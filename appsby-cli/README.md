# Usage
<!-- usage -->
```sh-session
$ npm install -g appsby-cli
$ appsby COMMAND
running command...
$ appsby (-v|--version|version)
appsby-cli/0.0.0 win32-x64 node-v12.18.1
$ appsby --help [COMMAND]
USAGE
  $ appsby COMMAND
...
```
